# 単語帳アプリ (名前募集中)

直感的に操作できる、Webベースの単語帳作成・共有アプリです。
中高生から社会人まで、自分だけの単語帳を素早く作成し、公開・共有することができます。

## 🚀 主要機能

- **単語帳作成・編集**: シンプルなインターフェースで、表裏のカードを簡単に追加できます。
- **CSV連携**: お手持ちのCSVファイルを読み込んだり、作成したデータをCSVで保存したりできます。
- **公開・共有機能**: 作成した単語帳を「公開」設定にすると、他のユーザーが検索して学習に利用できます。
- **ユーザープロフィール**: 他のユーザーの作成した単語帳一覧を簡単にチェックできます。
- **完全レスポンシブ**: PCはもちろん、スマートフォンやタブレットでも快適に動作します。

## 🛠 技術構成

- **フロントエンド**: 
    - HTML5 / CSS3 (Vanilla CSS)
    - JavaScript (ES6+, Firebase SDK v9 compat)
- **バックエンド / データベース**:
    - Firebase Authentication (ユーザー認証)
    - Firebase Cloud Firestore (リアルタイムデータベース)

## 📖 使いかた

1. **ログイン**: 登録・ログインをして自分専用のスペースを作成します。
2. **作成**: 「作成」ページから新しい単語帳のタイトルを決めます。
3. **編集**: 単語を一つずつ入力するか、CSVファイルを読み込んで一気に追加します。
4. **学習**: 作成した単語帳を開いて、フラッシュカード形式などで学習できます。
5. **公開**: 設定を「公開」にすると、他のユーザーがあなたの単語帳を見つけられるようになります。
